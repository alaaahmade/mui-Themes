## Installation

- In any folder, open your terminal
- write git clone https://github.com/alaaahmade/mui-Themes.git
- Then cd gaming-store
- run npm i
- run npm run dev
- open http://localhost:3000/ in your browser
```NOTE: Check out example.env```
